    Flipnotehax does not work/crashes when I try to use it!

    Instructions

    Section I - Files
    1. Go to the guide page where the flipnotehax files are located
    2. Copy the newest version of flipnotehax onto the sd card
    3. If files ask to merge, let them!
      4. Section II - Using the exploit
      1. Find the page on the site with installation
      2. Folow the instructions EXACTLY.
      3. If you still have issues, check that your SD card is not corrupted.
      4. If needed, contact the DSi Homebrew discord.

      I took out my SD Card. My game still freezes!

      HiyaCFW gives me an error!

      If the error says "FAT INIT FAILED", please redownload the files and check your SD card for errors.

      1. It is unknown at this time what to do if you get a generic error message.
      2. If you do, it is recommended that you power down you DSi, delete all files, and start over again.
