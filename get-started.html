    Get Started

    If you need help, ask the NDS(i)Brew Scene Discord server!
    If you would like to donate, you can do so through PayPal.

    This page will assist you in preparing a stock DSi for homebrew access. We will now install some generic tools that are needed for both a 1.4 downgrade (in preparation for RocketLauncher), and for setting up a permanent homebrew entrypoint.

    Before starting, Windows users should enable the option to show file extensions using this guide.

    Before starting, it's a good idea to check your SD card for errors using H2testw (Windows), F3 (Linux), or F3X (Mac).

    This will not work with Chinese DSi systems (A.K.A. iQue DSi) or Korean DSi systems as none of the exploitable apps have been released there. This guide, for the time being, is also incompatible with European DSi systems, but will be when ugopwn for EU is released.

    Flipnote Studio must be installed on your DSi system for this to work. Due to ugopwn only being released for USA currently, a United States system is additionally required.

    Instructions

    What you need

    • The latest release of ugopwn
    • The latest release of twlnf
    • The latest release of Python 3
    • The latest release of DSi SRL Extractor (with console_id.py)

    Section I - Preparation

    1. Power on your device.
    2. Go to System Settings > Data Management > System Memory > Flipnote Studio (or any other DSiWare) > Copy > Yes, and then exit to the DSi Menu. (If Data Management isn't appearing, open the DSi Shop, close it, and try again.)
    3. Power off your device, and insert the device's SD card into your computer.
    4. Copy the twlnf 1.4 installation script for your region to the root of your SD card.
    5. Copy the contents of ugopwn.zip to the root of your SD card.
    6. Copy the contents of twlnf-(version).7z to the root of your SD card, and rename twlnf.nds to boot.nds.
    7. Copy the contents of dsi_srl_extract.zip to a folder on your computer. Open your SD Card's drive, and navigate to Private > DS > Title. Copy your .bin file to your SRL Extractor directory. If multiple .bin files are present, only one needs to be copied.
    8. Open your SRL Extractor folder and double-click on console_id.py to run it. When prompted, press Enter. You will now see a console_id.txt file in the directory. (Note: On Unix-based systems (Mac and Linux, for example), WINE is required for this script to run.)
    9. Copy console_id.txt to the root of your SD card.
    10. Remove the SD card from your computer, and insert it into the SD card slot on your DSi.

    If you are interested in downgrading your DSi's firmware to v1.4 to prepare for RocketLauncher, continue to Installing v1.4.

    Otherwise, if you would like to install a permanent homebrew entrypoint, continue to Installing a Permanent Entrypoint.
